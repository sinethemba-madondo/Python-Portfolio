# Concert Ticket Booker

## Description

This Python program simulates a digital ticket booking system for concerts and live performances.

Users enter their name, the artist they would like to see, and the performance date. The program then generates a personalised booking confirmation message.

---

## Features

- Collects the user's full name
- Collects the artist or band name
- Collects the performance date
- Displays a personalised booking confirmation
- Uses formatted output with f-strings

---

## Concepts Used

- Variables
- Strings
- User Input
- f-strings
- Output Formatting
