# Secure Password Hint Tool

## Description

This Python program creates a simple username from the user's name and generates a secure password hint without revealing the actual password.

The project demonstrates Python string manipulation, indexing, user input, and formatted output while encouraging good password security practices.

---

## Features

- Collects the user's first name
- Collects the user's last name
- Automatically generates a username
- Accepts a secret password
- Displays a password hint using only the first and last letters
- Converts the password hint letters to uppercase for better visibility

---

## Concepts Used

- Variables
- Strings
- User Input
- String Indexing
- f-strings
- `.strip()`
- `.upper()`

---

## Technologies
