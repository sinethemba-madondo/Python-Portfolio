# Username & Message Formatter

## Description

This Python program collects a user's name and biography, formats the information, and generates a username automatically.

This project demonstrates Python string manipulation techniques and user input handling.

---

## Features

- Collects first and last name
- Formats the full name into Title Case
- Generates a lowercase username
- Removes extra spaces from the biography
- Counts the number of characters in the biography
- Replaces words in the biography using `.replace()`

---

## Concepts Used

- Variables
- Strings
- User input
- f-strings
- `.title()`
- `.lower()`
- `.strip()`
- `.replace()`
- `len()`
