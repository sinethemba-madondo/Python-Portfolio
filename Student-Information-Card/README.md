# Student Information Card

## Description

This Python program collects basic personal information from the user and displays it in a formatted profile card.

The project demonstrates fundamental Python programming concepts including user input, data types, arithmetic calculations, string formatting, and output presentation.

---

## Features

- Collects the user's first name
- Collects the user's last name
- Collects the user's age
- Collects the user's favourite number
- Calculates the user's age in months
- Rounds the favourite number to two decimal places
- Displays a personalised welcome message
- Formats text using string methods
