# MultiFunction Calculator

## Description

This Python program is a multifunction calculator that performs six mathematical operations using two numbers entered by the user.

The calculator demonstrates Python fundamentals such as arithmetic operators, conditional statements, user input, type conversion, rounding values, and basic error handling.

---

## Features

- Accepts two numbers from the user
- Performs addition
- Performs subtraction
- Performs multiplication
- Performs division
- Performs floor division
- Performs modulus
- Rounds numerical results to two decimal places
- Prevents division by zero by displaying a friendly error message

---

## Example Output

```text
Enter your first number: 25
Enter your second number: 4

RESULTS

Addition: 29.0
Subtraction: 21.0
Multiplication: 100.0
Division: 6.25
Floor Division: 6.0
Modulus: 1.0
```

If the second number is zero:

```text
Enter your first number: 25
Enter your second number: 0

Sorry, division by 0 is not allowed

RESULTS

Addition: 25.0
Subtraction: 25.0
Multiplication: 0.0
Division: Error
Floor Division: Error
Modulus: Error
```
